# Metode-avansate-de-extragere-a-semnalului-vocal-din-zgomot
Proiect MATLAB pentru reducerea zgomotului din semnale audio folosind STFT și spectral subtraction. Sistemul adaugă zgomot la diferite valori SNR, estimează zgomotul din cadrele inițiale, îl elimină în domeniul frecvență și reconstruiește semnalul folosind ISTFT.
